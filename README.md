## Features
- Write a daily note
- Put a mood on your note
- Give some tags to your note
- Search note by date
- Put your favorite note in special page

## Tech Stack
- Design Pattern : MVVM
- Local Database : Room
- Crashlytics & Analytics : Firebase
- UI Library : Android Material Design
